# iThinkAI 隐私政策 / Privacy Policy

**最后更新 / Last Updated: 2026-04-18**

---

## 中文版

### 1. 概述

感谢你使用 iThinkAI（"本插件"）。我们极其重视你的隐私。本插件的设计原则是：**尽可能少地接触你的数据，所有数据都留在你本地。**

### 2. 我们收集什么

**我们不收集任何数据。**

本插件不会将你的任何信息上传至任何服务器。插件中涉及的所有数据处理都发生在你本地浏览器内。

### 3. 本地存储的数据

为了提供功能，本插件会在你的浏览器本地（通过 Chrome `chrome.storage.local` API）存储以下信息：

- **插件设置**：倒计时秒数、启用的平台等
- **思考历史记录**：你在思考弹窗中主动输入的问题和想法，以及对应的时间戳、平台名称
- **使用统计**：思考次数、累计思考时长等数字统计

以上数据仅存储在你的浏览器中，**从不被传输到任何外部服务器**。你可以在插件设置中随时清空这些数据。

### 4. 插件读取的数据

为了实现核心功能，本插件会：

- **读取你在 AI 输入框中正在撰写的消息内容**：仅用于在思考弹窗中展示你即将发送的问题。此数据不会离开你的浏览器。

### 5. 网络请求

**本插件不会发起任何网络请求。** 所有逻辑均在本地执行。

### 6. 第三方

本插件不使用任何第三方分析服务、广告服务、追踪器或 SDK。

### 7. 数据共享

**我们不与任何第三方共享你的数据**，因为我们本身就没有收集你的数据。

### 8. 儿童隐私

本插件不专门面向 13 岁以下儿童，也不会从任何用户那里收集数据。

### 9. 政策变更

如本政策有变更，我们会在本页面更新"最后更新"日期，并在插件内提示。

### 10. 联系我们

如有任何隐私问题，请通过 Chrome Web Store 的反馈渠道联系我们，或通过本插件 GitHub 仓库的 Issues 提交。

## English Version

### 1. Overview

Thank you for using iThinkAI (the "Extension"). We take your privacy extremely seriously. This Extension is designed with one principle: **touch as little of your data as possible, and keep everything local.**

### 2. What We Collect

**We do not collect any data.**

The Extension does not upload any information to any server. All data processing happens locally in your browser.

### 3. Data Stored Locally

To provide functionality, the Extension stores the following data in your browser's local storage (via Chrome's `chrome.storage.local` API):

- **Extension Settings**: countdown duration, enabled platforms, etc.
- **Thinking History**: the questions and thoughts you voluntarily enter into the thinking modal, along with timestamps and platform names.
- **Usage Statistics**: session counts, cumulative thinking time, etc.

This data is stored **only in your browser** and is **never transmitted to any external server**. You can clear this data at any time via the Extension settings.

### 4. Data the Extension Reads

To implement core features, the Extension reads:

- **The message you are composing in the AI input field**: only to display it in the thinking modal. This data never leaves your browser.

### 5. Network Requests

**The Extension makes no network requests.** All logic runs locally.

### 6. Third Parties

The Extension uses no third-party analytics, advertising, trackers, or SDKs.

### 7. Data Sharing

**We do not share your data with any third party**, because we do not collect it in the first place.

### 8. Children's Privacy

The Extension is not directed at children under 13, and it does not collect data from any user.

### 9. Policy Changes

If this policy changes, we will update the "Last Updated" date on this page and notify users within the Extension.

### 10. Contact

For any privacy concerns, please reach out via the Chrome Web Store feedback channel or the GitHub repository Issues for this Extension.
